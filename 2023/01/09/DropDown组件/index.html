Unity DropDown组件

发表于2023-01-09|更新于2024-04-01|/Unity
|阅读量:

前段时间遇到了下拉栏的制作，之前没有用过DropDown组件，相关的组件属性以及api都不熟悉，趁此记录一下。

组件item

对象 作用
Label 初始化显示
Arrow 初始化下拉箭头
Template 准备生成的下拉栏模板
Item 滑动栏内的item模板
Item Background item背景图片
Item Checkmark item选中图片
Item Label item显示文本
Scrollbar 滚动栏（超出模板高度或宽度启用）

组件属性

属性名称 作用
Template 目标模板
Caption Text 下拉栏文本，可由选中的item控制
Caption Image 下拉栏图片，可由选中的item控制
Item Text item模板中的文本显示
Item Image item模板中的图片显示
Value 当前选中的索引
Alpha Fade Speed 下拉栏渐显渐隐速度
Options 下拉栏选项列表
On Value Changed 监听事件，Value改变时响应事件

常用相关API

item模板的text组件
